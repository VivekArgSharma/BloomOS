Create a visually breathtaking premium landing page for a futuristic AI-powered plant care platform.

The experience should combine:

* luxury editorial aesthetics,
* futuristic organic technology,
* Scandinavian minimalism,
* Apple-level cleanliness,
* cinematic typography,
* and calm wellness branding.

The final website should feel like:
“A luxury AI ecosystem designed for intelligent calm living.”

==================================================
CORE DESIGN THESIS
==================

This is NOT just a plant website.

The interface should feel like:

* a premium AI product,
* a futuristic wellness ecosystem,
* a calm productivity experience,
* a luxury lifestyle brand,
* and a modern editorial showcase.

The entire UI language must communicate:

* calmness,
* intelligence,
* softness,
* trust,
* simplicity,
* premium quality,
* elegance,
* and emotional warmth.

The design should feel:

* visually breathable,
* highly curated,
* emotionally calming,
* futuristic but human,
* organic yet technological.

==================================================
VISUAL PHILOSOPHY
=================

The design system should use:

* extreme whitespace,
* oversized cinematic typography,
* muted organic colors,
* floating UI cards,
* glassmorphism,
* soft gradients,
* subtle shadows,
* ambient lighting,
* asymmetrical balance,
* and layered composition.

Everything should feel:

* soft,
* intentional,
* spacious,
* premium,
* and alive.

No harsh geometry.
No corporate feeling.
No generic SaaS structure.

The layout should feel editorial and artistic rather than template-based.

==================================================
TYPOGRAPHY SYSTEM
=================

Typography is the PRIMARY identity of the interface.

Use contrast between:

* geometric sans-serif,
* and elegant editorial serif.

PRIMARY FONT OPTIONS:

* Satoshi
* General Sans
* Neue Montreal
* Inter

SECONDARY SERIF OPTIONS:

* Canela
* Cormorant Garamond
* Playfair Display
* Bodoni Moda

Typography should create:

* modern + natural,
* technology + elegance,
* simplicity + sophistication.

==================================================
TYPOGRAPHY RULES
================

HERO HEADING:

* gigantic scale
* cinematic presence
* ultra clean spacing
* thin-to-medium font weight
* tight line height
* luxurious kerning

Suggested:
font-size: 96px – 140px
line-height: 0.9 – 1
font-weight: 300 – 500

STRUCTURE EXAMPLE:

“Proper care of”
→ geometric sans-serif

“Plants”
→ elegant luxury serif

BODY TEXT:

* small
* soft gray
* readable
* elegant spacing

Suggested:
14px – 16px

==================================================
LAYOUT PHILOSOPHY
=================

Avoid rigid grid-heavy layouts.

Use:

* floating positioning,
* layered composition,
* visual anchoring,
* overlapping elements,
* soft asymmetry,
* editorial spacing rhythm.

The page should feel naturally composed.

==================================================
MAIN STRUCTURE
==============

Create:

* one massive floating rounded container,
* centered on the screen,
* with soft ambient shadow,
* large internal spacing,
* premium breathing room.

Inside the hero section:

* huge typography at top,
* plant/product visual in center,
* floating informational cards around object,
* supporting content block on side,
* subtle ambient shapes in background,
* elegant visual hierarchy.

The composition should create natural eye movement.

==================================================
IMAGE TREATMENT
===============

Use:

* isolated PNG imagery,
* realistic lighting,
* soft shadows,
* center-focused composition,
* premium cutout visuals.

The product/plant image should feel alive and elegant.

==================================================
UI COMPONENTS
=============

BUTTONS:

* pill shaped
* matte black or deep olive
* no harsh borders
* subtle hover lift
* smooth transitions

Style example:
border-radius: 999px;
padding: 14px 28px;
font-weight: 500;

Hover:
transform: scale(1.03);

==================================================
FLOATING INFO PILLS
===================

Floating cards should use:

* glassmorphism,
* soft borders,
* translucent surfaces,
* blurred backgrounds.

Style example:

background: rgba(255,255,255,0.7);
backdrop-filter: blur(10px);
border: 1px solid rgba(0,0,0,0.05);

Cards should:

* softly float,
* slightly glow,
* feel lightweight,
* feel futuristic.

==================================================
MOTION DESIGN
=============

Animations should be:

* slow,
* organic,
* subtle,
* cinematic,
* almost invisible.

Use:

* floating plant animation,
* staggered fade-ins,
* smooth hover motion,
* gradient glow movement,
* gentle parallax,
* ambient breathing motion.

Example:

transform: translateY(-6px);
animation: float 6s ease-in-out infinite;

Animation timing should feel elegant and premium.

Avoid flashy movement.

==================================================
BACKGROUND EFFECTS
==================

Add:

* blurred radial gradients,
* organic ambient blobs,
* soft glows,
* atmospheric lighting,
* translucent overlays,
* subtle layered textures.

Decorative elements should stay low opacity.
